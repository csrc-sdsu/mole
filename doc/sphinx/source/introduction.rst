Introduction
=================

MOLE (Mimetic Operators Library Enhanced) is a C++ and MATLAB library that implements mimetic finite difference methods. The library provides efficient implementations of fundamental differential operators and boundary conditions.

Key Features
-----------------

* Mimetic operators (Divergence, Gradient, Laplacian, Interpolation)
* Boundary conditions (Mixed, Robin)
* Support for both C++ and MATLAB interfaces
* Efficient implementations for 1D, 2D, and 3D problems
