# Advection Equation

This example demonstrates how to solve the advection equation using the MOLE library.

## Problem Description

The advection equation is given by:

$$\frac{\partial u}{\partial t} + \mathbf{v} \cdot \nabla u = 0$$

where $\mathbf{v}$ is the velocity field.

In this example, we solve the advection equation on the unit square [0,1]×[0,1] with appropriate boundary conditions.
