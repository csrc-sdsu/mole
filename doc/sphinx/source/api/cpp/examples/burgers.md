# Burgers Equation

This example demonstrates how to solve the Burgers equation using the MOLE library.

## Problem Description

The Burgers equation is given by:

$$\frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = \nu \frac{\partial^2 u}{\partial x^2}$$

where $\nu$ is the viscosity coefficient.

In this example, we solve the Burgers equation on the interval [0,1] with appropriate initial and boundary conditions.
