# Wave Equation

This example demonstrates how to solve the wave equation using the MOLE library.

## Problem Description

The wave equation is given by:

$$\frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u$$

where $c$ is the wave speed.

In this example, we solve the wave equation on the unit square [0,1]×[0,1] with appropriate initial and boundary conditions.
